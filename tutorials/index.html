Bayesian Statistics Toolbox (BST) example notebooks

Check out this page for examples of using BST.

All of the data and images are from the website for John K. Kruschke's textbook Doing Bayesian Data Analysis: A Tutorial with R, JAGS, and Stan.

These notebooks are adaptations of Jordi Warmenhoven's excellent PyMC3 port of the textbook as well as one from the PyMC developers. They are meant as example use cases that show you how to use BST and take advantage of some of the new functionality afforded by PyMC 4.0 in combination with ArviZ.

More complete examples to come soon!
